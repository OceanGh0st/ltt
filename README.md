ltt - LittleTerminalTasks

USAGE: ltt [options] [task/list name/ID]

	h, help prints out help
displaying options
	da	display all lists
	dl *	display list
moving options
	mv * ^  move task to list
adding options
	al *	add list
	at *	add task
removing options
	ra	remove all lists
	rl *	remove list
	rt *	remove task
	cl *	clean list


# move task
(ltt) da
